# vite-react-mobx-admin
vite react hook mobx antd 搭建权限管理后台系统


## 项目描述
使用 vite 作为构建工具, react+mobx+antd

## vite 配置详见 vite.config.ts
全部都有注释

## 目录结构

```
api > api 定义 
assets > 项目资源
common > 项目公共模块
components > 公共组件
hooks > 公共 hook
interface > 接口定义
logics > 逻辑
libs > 公共库
route > 路由
service > 数据处理
store > 数据
view > 页面
wrappers > 路由拦截
main.tsx > 入口
```

> 构建项目
```
yarn 
yarn start
```
